# Fahmi Prayuda — Portfolio

Modern frontend developer portfolio built with Next.js, Tailwind CSS, and Framer Motion.

🔗 **Live Demo:** ---
📂 **Repository:** https://github.com/fahmiprayuda/myportfolio  

---

## ✨ Overview

This portfolio showcases my work, projects, and frontend development skills.  
It focuses on clean architecture, smooth animations, responsive design, and scalable structure using feature-based organization.

---

## 🛠 Tech Stack

- **Next.js (App Router)**
- **React**
- **Tailwind CSS**
- **Framer Motion**
- **Vercel Deployment**

---

## 🚀 Features

- Modern UI with gradient & glassmorphism design
- Smooth animations using Framer Motion
- Filterable projects section
- Dynamic project detail pages (`/projects/[slug]`)
- SEO-ready metadata per page
- Responsive layout for all devices
- Clean feature-based architecture
